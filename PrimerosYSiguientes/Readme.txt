Este contiene la gramática, se debe dejar la primera línea vacía, y escribir a partir de la segunda línea, con una
línea vacía al final y espacios entre cada elemento de las gramática. (Se adjunta el primer ejemplo ya).
De ahí el resultado aparecerá en el archivo EjemploF.txt
